AIが「使えるかどうか」は、人間側の「使い方」で決まります。そう語るのは、グーグル、マイクロソフト、NTTドコモ、富士通、KDDIなどを含む600社以上、のべ2万人以上に思考・発想の研修をしてきた石井力重氏です。そのノウハウをAIで誰でも実践できる方法をまとめた書籍『AIを使って考えるための全技術』が発売されました。全680ページ、2700円のいわゆる“鈍器本”ながら、「AIと、こうやって対話すればいいのか！」「値段の100倍の価値はある！」との声もあり話題になっています。思考・発想のベストセラー『考具』著者の加藤昌治氏も全面監修として協力し、「これを使えば誰でも“考える”ことの天才になれる」と太鼓判を押した同書から、AIの便利な使い方を紹介します。

## AIを使った「様々な切り口」での発想法

AIを仕事に活用できるシーンは多々ありますが、業務の効率化や自動化だけに使うのは少々もったいないでしょう。新しいアイデアを考えるといった、「頭を使う作業」にもAIは活用できます。

ただし、適当な聞き方をしても、質の良い回答は得られません。ロクでもない回答が返ってきてしまうときには、人間側の質問（プロンプト）が適切でないことがほとんどです。

たとえば、様々な切り口でアイデアを生み出したいときにおすすめなのが、技法その17「**9つの型**」です。

これが、そのプロンプトです。

<smtcmp_block language="markdown">
```
〈アイデアを得たい対象を記入〉のアイデアをSCAMPER法に基づいて提案してください。以下の観点を参考にしてください。→S（代替）：部分、人、材料、働き、プロセス/C（結合）：部分、目的、応用方法、材料/A（適用）：状況、モノ、行為、考え/M（修正）：色、外形、音、音声、意味合い/M（拡大・縮小）：高さ、重さ、サイズ、強度、頻度、複雑さ、価値/P（置き換え）：そのままで別の分野、一部を変えて新しい用途、別の市場/E（削除）：部分、機能、動き、負担、価値/R（並べ替え）：パターン、配置、組み合わせ、部品/R（逆）：順序、上下、内外
```
</smtcmp_block>

「**SCAMPER（スキャンパー）法**」をご存じでしょうか。これは、アイデアを出そうとするときの「アタマの動き」を整理し、それぞれの頭文字を取ってまとめたもの。考えるための手がかりとなる「最初の質問」が整理された、王道の発想法です。

*   **S**ubstitute（代替）
*   **C**ombine（結合）
*   **A**dapt（適用）
*   **M**odify（修正）
*   **M**agnify/Minify（拡大/縮小）
*   **P**ut to other use（置き換え）
*   **E**liminate（削除）
*   **R**earrange（並べ替え）
*   **R**everse（逆）

それぞれの質問を、お題となる商品やサービスなどに「当てはめて」いくことで、アイデアを発想していきます。そんなSCAMPER法をまとめてAIで実践するのが、技法「9つの型」です。1つのお題に対して、9つの質問を一気にぶつけます。

どんな課題にも使える技法ですが、とくに深く考え始めるタイミングがおすすめです。9つの視点から幅広くアイデアが出てきますから、お題が持っている可能性やポテンシャルを潰すことのない、効果的な思考のとっかかりが見いだせます。あるいは、どこから考え始めようか迷う、困ってしまう、そんなときにもおすすめです。AIは余計なことを考えずに、型に従ってアイデアを拡げてくれます。

## 実践例：ドローンの活用法

では、実践してみましょう。

ローンチしたばかりの新しい商品やサービスは、まだその先がどこに行くのかわかりません。良く言えば、あらゆる可能性がある状態。そういう状態の商材には技法「9つの型」が効きます。なぜなら問いかけの方向性が総合的だからです。「ドローン」を例に実践してみましょう。

<smtcmp_block language="markdown">
```
〈新しいドローン〉のアイデアをSCAMPER法に基づいて提案してください。以下の観点を参考にしてください。→S（代替）：部分、人、材料、働き、プロセス/C（結合）：部分、目的、応用方法、材料/A（適用）：状況、モノ、行為、考え/M（修正）：色、外形、音、音声、意味合い/M（拡大・縮小）：高さ、重さ、サイズ、強度、頻度、複雑さ、価値/P（置き換え）：そのままで別の分野、一部を変えて新しい用途、別の市場/E（削除）：部分、機能、動き、負担、価値/R（並べ替え）：パターン、配置、組み合わせ、部品/R（逆）：順序、上下、内外
```
</smtcmp_block>

まだその活用法が確立していない「ドローン」。アイデアを考えようにも、最初に切り口を与えた段階で、いずれかの可能性を潰してしまうリスクもあります。そんなとき技法「9つの型」を使えば、考えうるあらゆる可能性が明示されます。さて、どのくらいの可能性が見つかるのか。ワクワクしながら回答を待ちましょう。

---

> 新しいドローンのアイデアをSCAMPER法に基づいて考えてみましょう。各要素について、以下のように提案します。
>
> S（代替）
> *   部分の代替：従来のプラスチックや金属製の部品を、環境に優しいバイオコンポジット材料で代替します。
> *   人の代替：遠隔操作が必要な作業をAIによる自律飛行で代替し、人間の介入を最小限に抑えます。
> *   材料の代替：軽量化と強度向上のために、新しいナノ材料やカーボンファイバーを使用します。
> *   働きの代替：ドローンの使用を物資輸送や撮影から、災害時の救助活動や農薬散布などに拡大します。
> *   プロセスの代替：組み立て工程を3Dプリンティング技術に置き換えて、製造効率とカスタマイズ性を向上させます。
>
> C（結合）
> *   部分の結合：ドローンに環境モニタリングセンサーを組み込み、飛行中の空気質や温度をリアルタイムで測定します。
> *   目的の結合：写真撮影機能と物資配送機能を一つのドローンに統合し、多機能化を図ります。
> *   応用方法の結合：VR技術と組み合わせて、ドローン操作者がより直感的に操縦できるようにします。
> *   材料の結合：軽量化と耐久性の両方を実現するために、複数の新材料を組み合わせて使用します。
>
> A（適用）
> *   状況の適用：ドローンを都市部の交通監視や混雑管理に適用し、交通流を改善します。
> *   モノの適用：家庭用として、窓拭きや雨樋の清掃などの家事を助ける小型ドローンを開発します。
> *   行為の適用：スポーツトレーニングでの使用を想定し、選手の動きを追跡して分析するドローンを提案します。
> *   考えの適用：「ゼロウェイスト」の考えをドローン設計に取り入れ、廃棄物を最小限に抑えた製品を目指します。
>
> M（修正、拡大/縮小）
> *   色の修正：迷彩や環境に溶け込む色彩を採用し、自然観察や動物の撮影に適したドローンを開発します。
> *   外形の修正：折りたたみ可能なデザインを採用し、携帯性を高めます。
> *   音の修正：ノイズリダクション技術を用いて、飛行時の音を最小限に抑えます。
> *   高さの拡大・縮小：高度制限を設けることで、安全性を向上させながらプライバシー保護にも配慮します。
> *   サイズの拡大・縮小：用途に応じて、超小型から大型までのサイズバリエーションを提供します。
>
> P（置き換え）
> *   そのままで別の分野への置き換え：農業用ドローンを、海洋調査や森林監視にも適用する。
> *   一部を変えて新しい用途：カメラを高性能な温度センサーに交換し、火災発見や救助活動に特化したドローンを開発します。
> *   別の市場への置き換え：子ども向けの教育用ドローンを開発し、STEM教育に貢献します。
>
> E（削除）
> *   部分の削除：余分な装飾や非必須の機能を削除し、シンプルでコスト効率の良いドローンを設計します。
> *   機能の削除：高度な自律飛行機能を持たせることで、手動操縦機能を削減します。
>
> R（並べ替え、逆）
> *   パターンの並べ替え：ドローンの群れを使って、光のショーを行うなど、エンターテインメントの新たな形を提案します。
> *   配置の並べ替え：バッテリーとセンサーの位置を最適化し、飛行効率とバランスを改善します。
> *   逆の考え：通常は上空から地上を撮影するドローンを、逆に地上から上空や建物の高い部分を観察するために使用します。
>
> これらのアイデアは、新しいドローンの開発において独創性と実用性を両立させるための出発点となります。各提案は、特定のニーズや問題点に応じてさらに詳細に検討する必要があります。

---

あっという間に、26個のアイデアを手に入れることができました。しかも9つの視点それぞれから。

*   「人の代替」：まだまだ操作には練習が必要なドローンですが、「自律飛行＋音声命令」機能を積んだドローンが発売されたら、使う人が一段と増えそうです。
*   「行為の適用」：スポーツトレーニングでの使用を想定し、選手の動きを追跡して分析するドローン。なるほどです。スポーツ科学が発達するなかで選手をより効率的に鍛えるためには、観察記録と分析フィードバックが大事。重量がかさばる計測機器を選手の身体に装着した際の余計な負荷や転倒等のリスクを考えると、追跡型カメラであるドローンには優位性があります。

## アイデアの深掘り

この点を掘り下げてみましょう。掘り下げ方もいろいろありますが、AIが回答してきたアイデアに対して、同じ技法でさらに掘り下げる連続技もおすすめです。さらに技法「9つの型」を使ってみます。

<smtcmp_block language="markdown">
```
「行為の適用：スポーツトレーニングでの使用を想定し、選手の動きを追跡して分析するドローンを提案します」をさらにSCAMPER法で、拡げてみてください。
```
</smtcmp_block>

初回にSCAMPER法の詳しい定義をしているので、2回目以降は簡単に触れるだけで充分です。

---

> 「行為の適用：スポーツトレーニングでの使用を想定し、選手の動きを追跡して分析するドローン」のアイデアをさらにSCAMPER法で広げるために、以下のように展開してみます。
>
> S（代替）
> *   データ収集の代替：選手の動きだけでなく、心拍数や体温などの生理学的データも一緒に収集できるセンサーをドローンに組み込みます。
> *   分析ツールの代替：標準的なビデオ分析ソフトウェアの代わりに、AIを活用した高度なパフォーマンス分析ツールを使用します。
>
> C（結合）
> *   技術の結合：AR（拡張現実）技術と組み合わせて、選手がトレーニング中に実際のパフォーマンスと比較してフィードバックをリアルタイムで受け取れるようにします。
> *   機能の結合：ドローンに照明を組み込み、夜間や低光量の環境でもトレーニングの質を維持できるようにします。
>
> A（適用）
> *   異なるスポーツへの適用：このドローンをサッカー、バスケットボール、陸上競技だけでなく、スキーやスノーボードなどのウィンタースポーツにも適用します。
> *   教育への適用：スポーツ科学の学生が選手のパフォーマンス分析を学ぶための教材としても使用します。
>
> M（修正、拡大/縮小）
> *   デザインの修正：ドローンを選手にとってさらに目立たないようにデザインを修正し、トレーニング中の集中を妨げないようにします。
> *   機能の拡大：選手のパフォーマンスだけでなく、チーム全体の動きを同時に分析し、戦術的な改善点を提案できるようにします。
>
> P（置き換え）
> *   トレーニング方法の置き換え：従来のビデオ撮影や手動での分析に代わり、このドローンを用いた自動化されたデータ収集と分析を標準的なトレーニング手法とします。
>
> E（削除）
> *   不必要な機能の削除：ユーザーインターフェースをシンプルにし、選手やコーチが必要としない複雑な機能を削除します。
>
> R（並べ替え、逆）
> *   データ分析のプロセスを逆転：まず最終目標とするパフォーマンスの指標を設定し、現在のパフォーマンスから逆算して必要な改善点を特定します。
> *   フィードバックの提供方法を再配置：リアルタイムフィードバックと定期的な詳細レポートの組み合わせを最適化し、選手とコーチのニーズに合わせて調整します。
>
> このようにSCAMPER法を用いてアイデアを展開することで、スポーツトレーニングにおけるドローンの使用をより革新的で包括的なものにすることができます。これらの提案は、選手のパフォーマンス向上、コーチングの質の向上、トレーニング方法の革新に貢献する可能性を秘めています。

---

*   「データ収集の代替：選手の動きだけでなく、心拍数や体温などの生理学的データも一緒に収集できるセンサーをドローンに組み込みます」。これは一般的なドローン開発がコストをかけるジャンルではなく、特定のスポーツに特化して、ユーザーのバイタルデータを非接触で観察・取得できるように発展させていくアイデア。新しいドローンとして面白そうです。
*   スポーツに限定せず、高齢者の散歩に随伴して体調急変をモニターしたり、予測技術で散歩の中止を提案したりする、「高齢者の体調モニタリングドローン」にも展開できそうですね。
*   他にも炎天下で、熱中症の生徒がいないか察知できるドローン。教員の目視だけではケアしきれないところを、ドローンでカバーします。教員の負担を減らせるなら、台数としても充分な需要を見込めるかもしれません。

四方八方から考えてみると、自分では予想もできなかったアイデアが手に入ります。
