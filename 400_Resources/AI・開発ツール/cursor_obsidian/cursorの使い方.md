## 体験1：魔法のショートカット `Cmd + K` (Ctrl + K)

**「ピンポイントでコードを生成・修正する」**機能です。

1. `index.html` を開きます。
    
2. 中身をすべて選択（`Cmd + A`）して、**`Cmd + K`** （Windowsは `Ctrl + K`）を押してください。
    
3. 入力欄が出てくるので、日本語でこう打ち込んでみましょう。
    
    > 「これを、パステルカラーでオシャレなデザインのToDoリストに変えて。タスクの追加と削除ができるJavaScriptも入れて。」
    
4. **Enter** を押すと、AIが目の前でコードを書き換えてくれます！
    
5. 右側に出る **「Accept」 or 「Keep」** ボタンを押すと確定します。

## 体験2：最強の自動構築 `Cmd + I` (Ctrl + I) ※Composer

これが現在のCursorの目玉機能、**「Composer（コンポーザー）」**です。複数のファイルにまたがる修正を一気にやってくれます。

1. **`Cmd + I`** （Windowsは `Ctrl + Shift + I`）を押して、Composer画面を開きます。
    
2. 以下のように指示してみてください。
    
    > 「今のToDoリストに、ダークモードに切り替えるボタンを追加して。あと、デザインを整えるために `style.css` という別ファイルを作って、そこにスタイルを切り出して。」
    
3. **「Apply」** を押すと、`index.html` の修正だけでなく、**新しいファイル（style.css）まで自動で作成**してくれます。
    
4. すべて **「Accept all」** で受け入れましょう。

## 体験3：AIにコードを解説してもらう `Cmd + L` (Ctrl + L)

書かれたコードがどう動いているか、AIに聞いてみましょう。

1. よくわからないコード（例えばJavaScriptの部分）をマウスでドラッグして選択します。
    
2. **`Cmd + L`** を押すと、右側にチャット画面が開きます。
    
3. 「この部分の処理を、プログラミング初心者にわかるように例え話を使って解説して」と送ってみてください。
    
    > 丁寧な解説が返ってきます
