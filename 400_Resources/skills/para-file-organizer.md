# /para スキル 利用ガイド

## 概要

`/para` は Obsidian vault のファイルを [[PARA メソッド]] に基づいて自動分類するスキルです。
ドライランで分類案を確認してから実行するため、**承認なしにファイルが動くことはありません。**

---

## 起動方法

Claude Code のチャット欄に以下を入力するだけで起動します。

```
/para
```

起動後、以下の 2 点を確認されます。

1. **対象フォルダ** — `100_inbox/` や `vault全体` など整理したい範囲を指定
2. **frontmatter 補完** — frontmatter のないファイルに `title / created / tags` を自動追加するかどうか

---

## 基本的な使い方

### 例 1: inbox を整理する

```
/para
→ 対象フォルダ: 100_inbox/
→ frontmatter 補完: はい
```

### 例 2: vault 全体をスキャンする

```
/para
→ 対象フォルダ: vault全体（すべてのフォルダ）
→ frontmatter 補完: いいえ
```

---

## 実行フロー（ステップ解説）

```
1. スキャン
   └─ Glob / Read でファイルを読み取り、PARA の評価順序で分類判断

2. Dry-run 出力（ここでは何も動かない）
   ├─ 移動提案一覧（ファイル名・移動元・移動先・判断根拠）
   ├─ 要確認ファイルへの質問（曖昧な分類はユーザーへ確認）
   └─ フォルダ構成の改善提案（サブフォルダ追加・アンチパターン修正）

3. 承認
   ├─ 「はい」→ 全件実行
   ├─ 「一部変更」+ 詳細 → 修正後に再確認
   └─ 「いいえ」→ キャンセル（何も変更されない）

4. 実行
   └─ ファイル移動・フォルダ作成・frontmatter 補完

5. 完了サマリー
   └─ カテゴリ別件数・微妙だった判断・保留項目を報告
```

## PARA フォルダ構造（このvaultの場合）

| カテゴリ | フォルダ | 分類基準 |
|---------|---------|---------|
| Projects | `200_Projects/` | 終わりがある活動。ゴールと期限があるもの |
| Areas | `300_Areas/` | 継続維持する責任領域（健康・家計・ブランドなど） |
| Resources | `400_Resources/` | 将来役立つかもしれない参考資料・関心トピック |
| Archive | `900_Archive/` | 完了したもの・関心が消えたもの |

迷ったら **より actionable な側（上）に分類** するのが原則です。

---

## スキルが質問してくる場面

以下のケースは、スキルが自律判断せずにユーザーへ確認します。

- ファイルが **Project と Area の間で曖昧**（財務・法務・税務・期限関連など）
- スクリーンショットの目的がファイル名・内容から読み取れない
- 複数のアクティブな Project に属しうる内容

---

## アンチパターン自動検出

フォルダ構成の以下の問題も検出して改善を提案します（実行は承認後のみ）。

| 問題 | 例 | 提案内容 |
|-----|---|---------|
| 空の中間フォルダ | `200_Projects/2026/` が空 | 削除 |
| 1 件しかないカテゴリ | `Work/` の中が 1 フォルダだけ | 階層を 1 つ削減 |
| 曖昧名フォルダ | `Stuff/` `Misc/` | 具体名へリネーム |
| 先回り作成の空フォルダ | まだ始まっていない案件の箱 | 削除または保留 |
| 意味のない深い階層 | 3〜4 階層あるが各階層に意味なし | 階層削減 |

---

## 注意事項

- **wikilink は壊れません。** フォルダ間移動のみの場合、Obsidian はファイル名ベースでリンクを解決するため自動的に維持されます。
- **ファイル名を変更する場合は別途確認が入ります。** vault 全体への影響があるため事前に通知されます。
- **バックアップは git で管理してください。** このスキルはバックアップを作成しません。実行前に `git commit` しておくことを推奨します。

`.claude/commands/para.md` に定義されています。
動作を変えたい場合はそのファイルを直接編集してください。
