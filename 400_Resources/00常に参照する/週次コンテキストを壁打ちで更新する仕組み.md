# 週次コンテキストを壁打ちで更新する仕組み（マニュアル）

> `000_me/02_context.md`（今この瞬間の自分の状態をAIに伝える週次ファイル）を、AIとの**壁打ち（1問ずつの対話）**で更新するための仕組み。
> Claude Code / Cursor / Antigravity CLI のどれでも同じように使える。
> 更新は**不定期・週1程度でOK**。思い立ったときに叩く運用。

---

## 1. これは何か

`02_context.md` は、AIに「今週の自分」を伝えるための週次ファイル。だが手で書こうとすると後回しになり、古くなりがち。

そこで、**AIに壁打ち相手をさせ、対話の流れで自動的にファイルを書き換える**仕組みを用意した。AIが前回の内容を踏まえて1問ずつ質問し、こちらが答えるだけで最新化される。まだ言語化できていない「気になっていること」も問いかけで引き出される。

---

## 2. 仕組みの構成（ファイルは2つ）

```
000_me/壁打ち_02_context更新.md      ← 正本（全ツール共通の壁打ちプロンプト）
.claude/commands/context-update.md   ← Claude Code 用の呼び出し口（正本を読むだけ）
        ↓ 書き換える対象
000_me/02_context.md                 ← 週次コンテキスト本体
```

- **正本は `000_me/壁打ち_02_context更新.md` の1ファイルだけ。** ここに全ロジックが書いてある。
- Claude のコマンドはこの正本を読むだけの薄いラッパー。だから**挙動を変えたいときは正本1か所を直せば3ツール全部に反映される**（二重管理しない）。

---

## 3. 使い方（特定の命令文）

| ツール | 起動方法 |
|---|---|
| **Claude Code** | `/context-update` |
| **Cursor** | `@壁打ち_02_context更新.md に従って 02_context を更新して` |
| **Antigravity CLI** | 同上（正本プロンプトファイルを参照させる言い回し） |

> Cursor / Antigravity はスラッシュコマンドの仕組みが違うため、「正本ファイルを参照させる」言い回しで起動する。中身（質問の流れ）はどのツールでも同じ。

---

## 4. 壁打ちの流れ

AIは次の順で、**1問ずつ**聞いてくる（まとめて列挙はしない）。各テーマで掘り下げの追い質問は最大1回。

1. **前回フォーカスの進捗** … 前回のメインフォーカスにどこまで進んだか／できなかった理由
2. **今週のメインフォーカス** … いちばん重心を置くこと
3. **サブタスク** … 具体タスク・ルーティン業務（福祉作業所など）
4. **気になっていること／頭の片隅** … 直接の作業ではないが引っかかっていること
5. **エネルギー・コンディション** … テンション／集中力／「これだけはやりたい」1つ（スキップ可）
6. **AIへの依頼スタンス** … 今週は壁打ち中心か、作業実行中心か

対話が終わると、AIが `02_context.md` を書き換え、**書き換え前に「この内容で反映してよいか」を確認**してくる。

---

## 5. 運用ルール（仕組みに組み込み済み）

- **捏造しない。** ファイルに書かれるのは自分が実際に言ったことだけ。特にコンディション欄は推測で埋めない。スキップすれば前回値か `（要記入）` が残る。
- **触れなかった欄は温存。** 答えなかったセクションは前回の内容がそのまま残る（勝手に消えない）。
- **週番号は自動。** ヘッダーは `YYYY年W{ISO週番号} / M/D〜M/D`（月曜始まり）で自動更新される。Cursor/Antigravity で日付が取れない場合は最初に日付を聞かれる。
- **頻度は手動・不定期。** 自動実行（cron 等）は設けていない。やりたいときに起動する。

---

## 6. メンテナンス

- **質問項目や口調を変えたい** → `000_me/壁打ち_02_context更新.md` を編集する（ここが正本）。Claude コマンドは触らなくてよい。
- **テンプレートの骨格を変えたい** → 同ファイル末尾の「02_context.md テンプレート」を編集する。
- **Claude のコマンド名を変えたい** → `.claude/commands/context-update.md` のファイル名を変える（ファイル名がコマンド名になる）。
