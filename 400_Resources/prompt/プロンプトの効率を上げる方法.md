---
## サンドイッチ構造

プロンプト冒頭は
```
# 役割
```
にするのが適している。
そして、プロンプトの最後は、
```
# やってほしいこと
```
という構成にして、真ん中に、参考になる物を入れる構成がいい。

---
## 目次

参考資料が長い場合は、
「この資料の目次を作成して」とAIに目次を作ってもらい、
資料の冒頭に貼り付けることで、AIの作業効率が上がる。

---
## 具体例

```
# 具体例
○○
```

フォーマットを指定するより、こちらの望む完成形の具体例を上記のような文章で貼り付けて、続けて

```
# 内容
○○

# やってほしいこと
○○
```

のような形で指示することで、こちらの望む回答を得ることが出来る。
