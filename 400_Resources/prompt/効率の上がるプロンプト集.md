## ゴールを一緒に考えて
```
ゴールの設定に自信がないから、壁打ち相手になって質問して。一気に全て質問するのではなく、ひとつずつ順番に質問して。
```

AIへの指示における鉄則は、「コンテキスト（背景）は丁寧に、プロンプトは最小に」です。

---
## 隠れ鵜呑み回避
```
この回答の「もっとよくできるところ」をTOP3挙げてください。また、それぞれの改善案も提示してください。
```

一見よさそうな回答の、内容の薄い部分をAIに自己批判させ、より具現性の高い回答へと導くプロンプト

---
## 小さなステップを踏みながら、確実に進める

```
一緒に考えてください。進め方の重要なお願いです。最初に「全体の工程」や「ToDoリスト」を提示しないでください。壁打ちしながら柔軟に考えたいので、「常に、現時点での最良のネクストアクション（問いかけ）」を1つだけ提示してリードしてください。先のことは決めず、私の返答を受けてから、また次にやるべきことを動的に判断してください。
```


こちらのチェックが及ばないくらい、先走らないよう、スモールステップを踏んでもらうプロンプト
