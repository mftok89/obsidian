# ChatGPTに自分の思考を邪魔されず、欲しい回答を導く「モノローグ法」とは？
https://www.youtube.com/watch?v=X8i7XJhJG94

### 批判的質問を引き出す
考えが行き詰ったときに、思考を広げるために使う。
耳の痛い話、気づいていなかった視点など、思考を深めるために、必要なことを、指摘してもらうことが出来ます。
 ```
 ここまでの私の思考プロセスを振り返ってください
 
 もしあなたが私の考えに懐疑的な立場の専門家だったらどんな批判的な質問を投げかけますか？
 
 以下の3つの視点から、それぞれ鋭い質問を1つずつ提示してください。
 
 １．論理的な矛盾や飛躍がないか
 ２．見落としているリスクや制約はないか
 ３．前提としている仮設は本当に正しいのか
 
 容赦なく本質を突く質問をしてください
 ```
### 多角的な視点を借りる
複数の専門家の視点を同時に手に入れる方法
```
ここまでの私の考えを３つの異なる専門家の視点から評価してください

１．マーケティング担当者の視点
 -市場性や顧客ニーズの観点からこの案をどう評価するか
 -どこを改善すべきか

２．財務担当者の視点
 -コスト構造や収益性の観点からこの案をどう評価するか
 -どんなリスクを懸念するか
 
３．エンジニアの視点
 -技術的な実現可能性の観点からこの案をどう評価するか
 -どこにボトルネックがあるか
 
それぞれの立場から瑞草区内券を述べてください 
 
```
