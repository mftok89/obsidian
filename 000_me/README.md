# 000_me の読み方

このフォルダは「私（熊澤 理）をAIに知ってもらい、効率よく開発に生かしてもらう」ためのフォルダ。
AIは新しいセッションでこのフォルダを参照するとき、まずこのファイルで各ファイルの役割を把握すること。

## ファイル一覧と役割

| ファイル | 役割 | 更新頻度 |
|---|---|---|
| [[00_profile]] | 不変に近いもの（バックグラウンド・価値観の要約） | ほぼ固定 |
| [[01_goals]] | 年間・四半期の目標 | 四半期ごと |
| [[02_context]] | 今週のフォーカス・コンディション | 週1（`/context-update` で更新） |
| [[03_preferences]] | AIとの協働スタイル（トーン・進め方） | ほぼ固定 |
| [[キーワード]] | 日常で感じた言葉の書き溜め場（インボックス） | 随時追記 |
| [[キーワード体系（私の行動指針）]] | **価値観・行動指針の正本** | キーワード整理時に更新 |
| [[壁打ち_02_context更新]] | `/context-update` の正本プロンプト | ほぼ固定 |
| [[自己紹介]] | 対外向け文書（文章素材） | 随時 |
| [[自分のポートフォリオ]] | 対外向け文書（mft.jp・提案資料の素材） | 随時 |

## キーワードの運用ルート

1. 日常で感じた言葉を [[キーワード]] の**末尾に追記**していく
2. 時々（月1目安）、`/keyword-update` で「反映済みライン以下の差分」を [[キーワード体系（私の行動指針）]] へ整理・反映する
   - 中間ファイル（まとめ等）は作らず、体系を直接更新する
   - 反映が終わったら、反映済みラインを末尾へ移動する（コマンドが自動で行う）
   - `/para` 実行時にも未反映キーワードがあれば知らせてくれる
3. 体系を読み返して、日々の行動を見つめなおす

## 補足

- 意思決定の記録は `300_Areas/AI協働/決断ログ.md` に集約（このフォルダにあった decisions_log は重複のため削除済み）
- 過去の中間生成物（キーワードまとめ等）は `900_Archive/000_me/` にある
